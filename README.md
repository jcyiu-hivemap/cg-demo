# Cenz × Grab — Interactive Prototype

A self-contained, clickable concept prototype showing what Cenz could look like built into the Grab app. Built for an internal walkthrough.

## View it

**Live:** once GitHub Pages is enabled, the prototype is at
`https://<your-username>.github.io/<repo-name>/`

**Locally:** just open `index.html` in any modern browser (Chrome, Safari, Edge). Works offline — all images are embedded, nothing to install.

## The flow

Tap the **Nutrition** tile on the Grab home screen, then follow the guided prompts:

1. **Cenz Home** — daily Health Score, focus rings, today's meals
2. **Order to hit your goal** — a recommended meal (Mr. Chow)
3. **GrabFood checkout** → **Place Order**
4. Back on the dashboard, the meal logs itself and the score moves — the loop closing

You can also tap the camera to snap a meal, or tap any logged meal to see its analysis. Use **Restart demo** to reset.

## Notes

- Single file (`index.html`); the food photos are embedded as data, so it's fully portable.
- Set to `noindex` so it won't appear in search results.
- Best viewed on a phone screen or a narrow browser window.
